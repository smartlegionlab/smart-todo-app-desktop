## Help:

- Clone project
- Go to the project folder
- `python -m venv venv`
- `pip install -r requirements.txt`
- `python main.py`

### Manjaro fix:

- `sudo pacman -S mpv`
- `sudo ln -s /usr/lib/libmpv.so /usr/lib/libmpv.so.1`
